# Plugin Installation Guide

This guide provides step-by-step instructions for installing and managing plugins on the Services AI platform.

## Installing a Plugin

1. **Access the Plugin Management Page**
   - Navigate to the Admin dashboard
   - Click on "Plugins" in the sidebar menu

2. **Upload a Plugin**
   - Click the "Upload Plugin" button
   - Select a plugin ZIP file from your computer
   - Click "Upload"

3. **Review Plugin Information**
   - Verify the plugin details (name, version, author, description)
   - Review the required permissions
   - Click "Install" to proceed

4. **Configure Plugin**
   - After installation, you'll be redirected to the plugin detail page
   - Enable the plugin using the "Enable" button
   - Configure any required settings
   - Enable necessary permissions

## Managing Plugins

### Enabling/Disabling a Plugin

1. Navigate to the Plugin Management page
2. Find the plugin you want to enable/disable
3. Click "Manage" to open the plugin detail page
4. Click the "Enable" or "Disable" button

### Configuring Plugin Settings

1. Navigate to the plugin detail page
2. Scroll down to the "Settings" section
3. Update the settings as needed
4. Click "Save Settings"

### Managing Plugin Permissions

1. Navigate to the plugin detail page
2. Find the "Permissions" section
3. Enable or disable individual permissions as needed
   - Note: Some plugins may not function properly if required permissions are disabled

### Uninstalling a Plugin

1. Navigate to the plugin detail page
2. Click the "Uninstall" button
3. Confirm the uninstallation
   - Warning: This will permanently remove the plugin and all its data

## Troubleshooting

### Plugin Won't Enable

- Check if all required settings are configured
- Verify that the plugin is compatible with your Services AI version
- Check the system logs for error messages

### Plugin Doesn't Appear on Dashboard

- Ensure the plugin is enabled
- Verify that the "dashboard_widget" permission is enabled
- Check if the plugin provides a dashboard widget implementation

### Plugin Settings Not Saving

- Ensure the input values meet any validation requirements
- Check for any JavaScript errors in the browser console
- Try refreshing the page and attempting to save again

## Plugin Security Guidelines

- Only install plugins from trusted sources
- Review plugin permissions carefully before enabling
- Disable permissions that aren't necessary for the plugin's functionality
- Keep plugins updated to the latest versions

## Getting Help

If you encounter issues with a specific plugin, contact the plugin author using the email provided in the plugin details. For platform-related issues, contact the Services AI support team.
